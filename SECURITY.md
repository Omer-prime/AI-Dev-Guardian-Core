# Security Policy

## Reporting vulnerabilities
If you discover a security issue in AI Dev Guardian, please do NOT open a public issue.

Instead, contact the maintainers privately:
- GitHub: open a private security advisory (preferred)

## Supported versions
We support the latest `main` branch and the most recent release tag.

## Disclosure
We aim to respond quickly and ship a fix as soon as possible.
