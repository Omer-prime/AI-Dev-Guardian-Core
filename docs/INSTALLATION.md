# Installation

AI Dev Guardian (Core) is a Rust-based tool. You can run it from source or install it locally.

## Option A — Install from source (fastest)
```bash
git clone https://github.com/Omer-prime/AI-Dev-Guardian-Core.git
cd AI-Dev-Guardian-Core
cargo install --path cli
ai-dev-guardian --help
